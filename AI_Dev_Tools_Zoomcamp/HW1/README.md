# HW1 – Django TODO App

AI Dev Tools Zoomcamp 2025 – Homework 1

## Overview

This is a simple TODO web application built with **Django** and managed with **Poetry**, created as part of the AI Dev Tools Zoomcamp (Homework 1).

The goal of this homework is to practice using AI assistants to build a small but complete web app:

- Set up a Django project and app  
- Define models and run migrations  
- Implement views and URL routing  
- Create HTML templates  
- Add tests and run them from the terminal  

## Tech Stack

- Python 3.12 (or compatible)
- Poetry
- Django 5.x
